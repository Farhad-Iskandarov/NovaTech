PROJECT OVERVIEW
--------------------------------------------------------------------------------
A full-stack, production-ready educational website for "Novatech Education Center" 
located in Sumgayit, Azerbaijan. The website supports multiple languages and 
includes a comprehensive admin panel for content management.

================================================================================
                              TECHNOLOGY STACK
================================================================================

FRONTEND:
- React.js (JavaScript)
- Tailwind CSS for styling
- Framer Motion for animations
- Shadcn/UI component library
- Axios for API calls
- React Router for navigation
- Recharts for admin dashboard charts

BACKEND:
- FastAPI (Python)
- MongoDB database
- Motor (async MongoDB driver)
- JWT authentication with bcrypt
- Pydantic for data validation

================================================================================
                              API ENDPOINTS
================================================================================

PUBLIC ENDPOINTS:
-----------------
GET  /api/                      - API info
POST /api/seed                  - Seed database with initial data
GET  /api/courses               - Get all courses
GET  /api/courses/:id           - Get course by ID
GET  /api/faqs/:course_id       - Get FAQs for a course
GET  /api/blogs                 - Get all published blogs
GET  /api/blogs/:slug           - Get blog by slug
GET  /api/testimonials          - Get active testimonials
GET  /api/teachers              - Get active teachers
POST /api/submissions/contact   - Submit contact form
POST /api/submissions/application - Submit course application
POST /api/analytics/pageview    - Track page view

AUTHENTICATION:
---------------
POST /api/auth/register         - Register new admin (disabled in production)
POST /api/auth/login            - Admin login
GET  /api/auth/me               - Get current user

PROTECTED ENDPOINTS (require JWT token):
----------------------------------------
POST   /api/courses             - Create course
PUT    /api/courses/:id         - Update course
DELETE /api/courses/:id         - Delete course

POST   /api/faqs                - Create FAQ
PUT    /api/faqs/:id            - Update FAQ
DELETE /api/faqs/:id            - Delete FAQ

POST   /api/blogs               - Create blog
PUT    /api/blogs/:id           - Update blog
DELETE /api/blogs/:id           - Delete blog

POST   /api/testimonials        - Create testimonial
PUT    /api/testimonials/:id    - Update testimonial
DELETE /api/testimonials/:id    - Delete testimonial

POST   /api/teachers            - Create teacher
PUT    /api/teachers/:id        - Update teacher
DELETE /api/teachers/:id        - Delete teacher

GET    /api/submissions         - Get all submissions
PUT    /api/submissions/:id/read - Mark as read
DELETE /api/submissions/:id     - Delete submission

GET    /api/analytics/summary   - Get analytics summary

================================================================================
                              HOW TO USE
================================================================================

1. ACCESSING THE WEBSITE:
   - Open the website URL in browser
   - Default language is Azerbaijani

2. CHANGING LANGUAGE:
   - Click on language switcher (flag icon) in header
   - Select desired language (Azerbaijani, English, Russian)

3. SWITCHING THEME:
   - Click sun/moon icon in header to toggle dark/light mode

4. VIEWING COURSES:
   - Navigate to "Kurslar" (Courses)
   - Use filters to narrow by category
   - Click "Ətraflı" (Learn More) for details

5. APPLYING FOR A COURSE:
   - Go to course detail page
   - Click "Müraciət Et" (Apply Now)
   - Fill form and submit

6. CONTACTING:
   - Go to "Əlaqə" (Contact) page
   - Fill contact form or use WhatsApp button

7. ADMIN ACCESS:
   - Go to /nova-admin
   - Login with admin credentials
   - Manage all content from dashboard
